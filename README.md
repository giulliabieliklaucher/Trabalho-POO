[RESUMO]
Este projeto consistem em um jogo de gerenciamento de recursos de um reino, o jogador deve conseguir manter o equilíbrio entre a população, igreja, exercito e dinheiro. Foi desenvolvido para a matéria de Programação Orientada a Objetos 1 do primeiro semestre do segundo ano do curso de ciência da computação.

[EXECUÇÃO]
1- Abrir o projeto na ide Intelij
2- Navegue até o arquivo Main.java
3- Aperte o botão Play para rodar a classe principal


[CONCEITOS APLICADOS]
Para a construção do projeto, foram utilizados os conceitos de programação Orientada a Objetos aprendidas durante o semestre, tais como: encapsulamento e ocultação de informação, herança, polimorfismo, interface, classes abstratas e, também, o uso de arquivos para saída de dados
